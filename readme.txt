1. Загрузите файлы из архива через любой ftp-клиент на свой сайт, в корень сайта: www, public_html и другие – зависит от хостинга.
2. Введите в строке браузера адрес: http://имя_вашего_сайта/install.php
3. Укажите параметры доступа к базе данных MySQL.
4. Нажмите "Установить" и подождите 1-2 минуты, пока автоматически загрузится и распакуется архив с программой.
5. Удалите файлы установщика install.php.
